🤖 Machine Learning Algorithms
🌲
Isolation Forest
Unsupervised
Detects anomalies without needing labeled data. Builds random decision trees and measures how easy it is to isolate each data point. Anomalies are isolated faster.
  • 1Randomly selects a feature and split value
  • 2Builds 100 isolation trees on the data
  • 3Counts how many splits needed to isolate a point
  • 4Few splits = anomaly. Many splits = normal
🧠
Random Forest
Supervised
100 decision trees each vote on the attack type. The majority vote wins. Trained on labeled examples of each attack type — gives exact classification.
  • 1Trained on 1,000 labeled cloud log events
  • 2Each tree learns different feature combinations
  • 3All 100 trees vote on attack type
  • 4Majority vote = final classification
🔵
K-Means Clustering
Unsupervised
Groups all traffic into 6 behavioral clusters. Each cluster represents a type of network behavior. Events that don't fit any known cluster pattern are flagged.
  • 1Places 6 random cluster centers in feature space
  • 2Assigns each event to nearest cluster center
  • 3Moves centers to mean of their assigned events
  • 4Repeats until clusters stabilize
