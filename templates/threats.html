{% if name == 'Brute Force' %}

🔍 How to Identify

  • Login attempts > 50 per session
  • Requests per minute > 200
  • Very short duration per request (<0.1s)
  • Status code 401 (Unauthorized) repeated
  • Single IP, multiple usernames

🤖 How AI Detects It

  • Isolation Forest: login_attempts outlier
  • Random Forest: classifies pattern as Brute Force
  • K-Means: falls into high-frequency cluster
  • Feature trigger: login_attempts × requests_per_min
{% elif name == 'DDoS' %}

🔍 How to Identify

  • Requests per minute > 5,000
  • CPU usage jumps above 85%
  • Bytes sent > 50,000 per event
  • Extremely short duration (0.001s)
  • Traffic from many IPs simultaneously

🤖 How AI Detects It

  • Isolation Forest: cpu_usage + requests anomaly
  • Random Forest: bytes_sent is top feature
  • K-Means: high-volume cluster segregation
  • Feature trigger: requests_per_min × cpu_usage
{% elif name == 'Data Exfiltration' %}

🔍 How to Identify

  • bytes_sent > 500,000 (massive outbound)
  • bytes_received very low (one-way flow)
  • Long duration (30–300 seconds per session)
  • Low requests per minute (slow, deliberate)
  • Happens at unusual hours

🤖 How AI Detects It

  • Isolation Forest: bytes_sent extreme outlier
  • Random Forest: bytes_sent/received ratio
  • K-Means: unique slow-large-transfer cluster
  • Feature trigger: bytes_sent × duration
{% elif name == 'Port Scan' %}

🔍 How to Identify

  • Very tiny packets (40–80 bytes sent)
  • Rapid connections (requests_per_min: 500–2000)
  • Near-zero duration per connection
  • Probing many different ports
  • Low CPU usage (lightweight probing)

🤖 How AI Detects It

  • Isolation Forest: tiny bytes + high requests anomaly
  • Random Forest: bytes_sent + duration pattern
  • K-Means: micro-packet cluster
  • Feature trigger: bytes_sent ÷ requests_per_min ratio
{% elif name == 'Insider Threat' %}

🔍 How to Identify

  • Access at unusual hours (midnight–4am)
  • Long session duration (60–600 seconds)
  • Moderate-high bytes transferred
  • Authorized user, unauthorized behavior
  • Unusual data access patterns

🤖 How AI Detects It

  • Isolation Forest: duration + time-of-day outlier
  • Random Forest: duration × bytes_sent pattern
  • K-Means: falls outside normal user clusters
  • Feature trigger: duration × bytes_sent
{% endif %}
